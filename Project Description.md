# 🧹 The Cleanic – Responsive Cleaning Service Website

## 📌 Project Overview

The Cleanic is a fully responsive and modern cleaning service website developed using **HTML5, CSS3, Bootstrap 5, JavaScript, and Font Awesome**. The project is designed to provide a professional online presence for a cleaning company with visually appealing layouts, smooth responsiveness, interactive functionality, and user-friendly sections.

The website features a premium and elegant interface with modern UI/UX design principles, proper spacing, alignment, responsive layouts, animations, and dynamic JavaScript interactions to enhance the overall user experience.
